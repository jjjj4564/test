## 開發設置

### 官網

cd 至 site-frontend
```
# 安裝套件
npm install

# 執行vue-cli-service
npm run dev

# 編譯檔案至專案根目錄的public
npm run build
```


### 報到功能頁
cd 至 checkin-plus-frontend
```
# 安裝套件
npm install

# 執行vue-cli-service
npm run dev

# 編譯檔案至專案根目錄的public
npm run build
```


### 後端

此專案須設置後端環境，直接詢問後端
